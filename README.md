# CodeHS Crawler

This is a NodeJS tool using the Puppeteer headless browser to crawl CodeHS.com for student grades.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install CodeHS Crawler.

```bash
npm install code_hs_crawler
```

## Usage

Run Main.java
